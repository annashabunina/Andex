## О BI-системе Power BI
Power BI — это мощный инструмент бизнес-аналитики от компании Microsoft, который позволяет пользователям визуализировать данные и делиться инсайтами в организации. Он предоставляет широкий спектр возможностей для анализа данных и создания интерактивных отчетов и панелей управления. Вот основные функции и возможности Power BI:

1. Импорт данных: Power BI поддерживает подключение к различным источникам данных, включая базы данных, Excel, облачные сервисы и API. Пользователи могут загружать данные из множества источников, что упрощает процесс анализа.
2. Моделирование данных: Инструмент позволяет создавать модели данных, определять отношения между таблицами и применять вычисляемые столбцы и меры с помощью языка DAX (Data Analysis Expressions). Это помогает в создании более сложных аналитических решений.
3. Визуализация данных: Power BI предлагает широкий выбор визуализаций, включая графики, диаграммы, карты и таблицы. Пользователи могут легко настраивать внешний вид визуализаций и комбинировать их в интерактивные отчеты.
4. Интерактивные отчеты: Отчеты в Power BI могут быть интерактивными, что позволяет пользователям взаимодействовать с данными, фильтровать их и исследовать различные аспекты информации.
5. Общий доступ и сотрудничество: Power BI позволяет пользователям делиться отчетами и панелями управления с коллегами и заинтересованными сторонами. Это может быть сделано через Power BI Service, где пользователи могут публиковать свои отчеты и управлять доступом.
6. Обновление данных: Power BI поддерживает автоматическое обновление данных, что позволяет пользователям получать актуальную информацию без необходимости вручную обновлять отчеты.
7. Анализ в реальном времени: Инструмент предоставляет возможность анализа данных в реальном времени, что особенно полезно для мониторинга ключевых показателей и оперативного принятия решений.
8. Интеграция с другими продуктами Microsoft: Power BI хорошо интегрируется с другими сервисами Microsoft, такими как Excel, Azure и Teams, что делает его удобным для пользователей экосистемы Microsoft.
9. Мобильные приложения: Power BI также предлагает мобильные приложения, которые позволяют пользователям получать доступ к отчетам и панелям управления на мобильных устройствах.
10. AI и машинное обучение: Power BI включает функции искусственного интеллекта, такие как автоматическое создание визуализаций, анализ аномалий и использование предсказательной аналитики.

В целом, Power BI является мощным инструментом для анализа и визуализации данных, который подходит как для небольших компаний, так и для крупных организаций, стремящихся улучшить свои бизнес-процессы на основе данных.

#### MAU, DAU, Stickiness
MAU (Monthly Active Users) — это количество уникальных пользователей, которые взаимодействовали с продуктом хотя бы раз в течение месяца.
DAU (Daily Active Users) — это количество уникальных пользователей, которые взаимодействовали с продуктом хотя бы раз в течение дня.
Stickiness — это показатель вовлеченности, рассчитываемый как отношение DAU к MAU, выраженное в процентах. Он показывает, насколько часто пользователи возвращаются к приложению.

```
_DAU = 
AVERAGEX(
    SUMMARIZE
    (
        'userentry',
        'calendar_progress'[Дата],
        "users",
        DISTINCTCOUNT('userentry'[user_id])
    ), 
    [users]
)
```
```
_MAU = 
AVERAGEX(
    SUMMARIZE(
        'userentry',
        calendar_progress[Год],
        'calendar_progress'[Месяц],
        "users", 
        DISTINCTCOUNT('userentry'[user_id])
    ), 
    [users]
)
```
```
_Stickiness = DIVIDE([_DAU], [_MAU])
```

#### Rolling Retention
Rolling retention (или "постоянная удерживаемость") — это метрика, которая измеряет процент пользователей, которые продолжают использовать продукт в течение определенного времени после их первого взаимодействия. Например, rolling retention может показывать, сколько пользователей, зарегистрировавшихся в приложении, вернулись через 7, 14 или 30 дней после регистрации. Эта метрика помогает понять, насколько эффективно приложение удерживает пользователей в долгосрочной перспективе.
```
Retention = 
(SUMMARIZE
    (userentry, 
    userentry[user_id], 
    "max_date", 
    MAX('userentry'[date_entry_at])
    )
)
```
```
dif = DATEDIFF(Retention[date_joined_at],Retention[max_date], DAY)
```
```
retention_3_day = IF('Retention'[dif] >= 3, 1,0)
retention_7_day = IF('Retention'[dif] >= 7, 1,0)
retention_14_day = IF('Retention'[dif] >= 14, 1,0)
retention_30_day = IF('Retention'[dif] >= 30, 1,0)
```
