# RFM-анализ для email-рассылок аптечной сети А.Аптека
А.Аптека - молодая аптечная сеть, которая входит в экосистему Андекс. Ввиду снижения спроса в теплый период года директор компании обратился к помощи аналитика. На данном этапе существует система email-рассылок. Но есть проблема: все рассылки для разных клиентов имеют одинаковое содержание, из-за чего работают неэффективно. 
**Цель исследования** – получить классификацию пользователей и представить стратегию для работы с email-рассылкой.
Для классификации клиентов был проведен RFM-анализ. Это метод анализа клиентской базы данных, используемый для сегментации клиентов на основе трех основных параметров:
- **Recency** (Недавность) - оценка времени, прошедшего с момента последней покупки клиента. 
- **Frequency** (Частота) - количество покупок, совершенных клиентом за определенный период времени. 
- **Monetary** (Денежный объем) - суммарная стоимость всех покупок, совершенных клиентом за определенный период времени. 

Чтобы использовать RFM-анализ, для начала была исследована база данных компании. Далее были рассчитаны метрики распределения величин: Среднее, минимум, максимум, медиана, 33-й перцентиль, 67-й перцентиль, децили.
После были подобраны границы значений для классификации. В итоге получилось 27 категорий. Однако для бизнеса будет слишком затратной уникальная рассылка для каждой группы. Поэтому категории были разделены на 8 групп. Получились следующие группы с процентом их встречаемости:
- VIP (6.17%)
- Постоянные (15.90%)
- Новички (11.14%)
- Спящие постоянные (7.69%)
- Спящие редкие (26.27%)
- Уходящие постоянные (2.16%)
- Уходящие редкие (7.62%)
- Одноразовые (23.06%)
  
Для каждой группы были даны рекомендации, которые включают в себя, например, информационные рассылки, рассылки с акциями, опросы, приглашения на различные мероприятия и т.д.
В целом, данный анализ был проведен успешно. Теперь менеджерам легче и эффективнее составлять содержание рассылок, что приведет к увеличению чека и частоте покупок.
